# 흑백요리사 맛집 지도

넷플릭스 흑백요리사 출연 셰프들의 레스토랑을 지도에서 찾아보세요.

## 기능

- 지도에서 셰프들의 레스토랑 위치 확인
- 시즌별 필터 (시즌 1 / 시즌 2)
- 팀별 필터 (흑수저 / 백수저)
- 레스토랑 상세 정보 (예약 방법, 가격대, 영업시간 등)
- 캐치테이블/네이버 예약 링크 연결

## 포함된 셰프

### 시즌 1 (2024)
- 안성재 (모수, 모수 한남)
- 나폴리 맛피아 (라탄티나, 팔레르모)
- 최현석 (쵸이닷, 앳홈키친)
- 여경래, 오세득, 정지선, 박준우
- 이영숙, 조희숙, 트리플스타
- 에드워드 리

### 시즌 2 (2025)
- 임기학 (밍글스, 막시무스)
- 선경 (순)
- 김도윤 (에빗)
- 장호준 (호야)
- 문기훈 (몽탄, 육회본가)

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- Leaflet (지도)
- OpenStreetMap
